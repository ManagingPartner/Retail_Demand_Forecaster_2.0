# Retail Demand Forecaster

## Overview
[cite_start]This project forecasts daily retail demand using a `RandomForestRegressor`[cite: 27]. [cite_start]It maintains a time-series feature pipeline to identify patterns like weekend effects and seasonality[cite: 26, 35].

## Usage
1. Ensure you have `pandas` and `scikit-learn` installed (`pip install pandas scikit-learn`).
2. Run the forecaster:
   ```bash
   python forecaster.py
